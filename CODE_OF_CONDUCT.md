# Code of Conduct

This project aims to be a respectful, practical space for maintaining a small privacy-sensitive tool.

## Expected behavior

- Be kind and direct.
- Assume good intent while staying precise about risks.
- Respect that exported notes and database files can contain highly private information.
- Keep issue discussions focused on reproducible behavior and safe, redacted examples.

## Unacceptable behavior

- Harassment, insults, threats, or discriminatory language.
- Posting another person's private note content, database files, credentials, or personal data.
- Pressuring maintainers or users to disclose private exports.

## Reporting

If you see a problem, open a private security report when sensitive data is involved, or contact the maintainers through the repository owner profile. Maintainers may remove content or block participants to protect users and contributors.
